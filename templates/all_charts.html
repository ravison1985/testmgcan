100-Day Charts User Guide
Overview

100-Day Charts page displays comprehensive candlestick charts with support/resistance levels from volume profile analysis for all F&O stocks.

Key Features
  • 100-Day Candlestick Charts: Historical price action with OHLC data
  • Volume Profile Levels: Support and resistance zones (VPOC, VAH, VAL)
  • Auto-Loading: Charts load progressively for optimal performance
  • Search Functionality: Quick search to find specific stocks
  • Responsive Grid: 2-column layout on desktop, 1-column on mobile
How to Use
  • Search: Type stock symbol in search box to filter charts
  • Clear Search: Click X button to show all charts
  • View Progress: Loading progress bar shows chart loading status
  • Scroll: Charts load as you scroll down the page
Understanding the Charts
  • Green Candles: Closing price higher than opening (bullish)
  • Red Candles: Closing price lower than opening (bearish)
  • Horizontal Lines: Volume profile support/resistance levels
  • VPOC: Point of Control - highest volume traded price level
  • VAH/VAL: Value Area High/Low - 70% volume concentration zone
